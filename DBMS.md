**DBMS (Database Management System)** is software that allows users to **create, store, retrieve, update, and manage data** in a database efficiently.

### Definition

A **Database Management System (DBMS)** is a software system that acts as an interface between the database and users/applications, ensuring organized data storage, security, consistency, and easy access.

### Functions of a DBMS

* Stores data in an organized manner.
* Inserts, updates, and deletes data.
* Retrieves data using queries.
* Ensures data security and integrity.
* Controls concurrent access by multiple users.
* Provides backup and recovery.

### Advantages

* Reduces data redundancy.
* Improves data consistency.
* Enhances data security.
* Enables data sharing among multiple users.
* Simplifies backup and recovery.

### Disadvantages

* High cost of software and maintenance.
* Requires skilled administrators.
* Performance may decrease for very large or complex systems.

### Examples of DBMS

* MySQL
* Oracle Database
* Microsoft SQL Server
* PostgreSQL
* SQLite

### Simple Example

Consider a **college database**:

* **Students Table:** Student_ID, Name, Department
* **Courses Table:** Course_ID, Course_Name
* **Enrollment Table:** Student_ID, Course_ID

The DBMS manages these tables and allows users to perform operations such as adding students, updating records, or finding which courses a student has enrolled in.

If you're preparing for exams, remember this one-line definition:

> **DBMS is software used to create, manage, store, retrieve, and control access to data in a database.**
